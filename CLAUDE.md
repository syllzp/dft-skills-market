## Project Overview

**dft-skills-market** — Portable AI skills market for DFT calculations (ORCA/VASP/QE/Gaussian/CP2K). Modular, self-contained, install-on-demand skill suites for input generation, error diagnosis, output parsing, and workflow automation for computational chemistry.

## Supported DFT Packages

- ORCA
- VASP
- Quantum Espresso (QE)
- Gaussian
- CP2K

## Skill Categories

- **Input generation** — Creating input files for DFT packages
- **Error diagnosis** — Detecting and explaining calculation errors
- **Output parsing** — Extracting results from DFT output files
- **Workflow automation** — Orchestrating multi-step DFT calculations
